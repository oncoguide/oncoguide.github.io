# OncoGuide

Practical, evidence-based cancer guidance for patients and families.

Free. Non-commercial. Always.

## About

OncoGuide is an educational oncology blog created by patients, for patients. It provides actionable, evidence-based information in plain language across 6 languages: English, Romanian, Italian, French, German, and Spanish.

## Built With

- [Hugo](https://gohugo.io/) — Static site generator
- [PaperMod](https://github.com/adityatelange/hugo-PaperMod) — Theme
- [GitHub Pages](https://pages.github.com/) — Hosting

## Development

```bash
# Run locally
hugo server --buildDrafts

# Build for production
hugo --gc --minify
```
